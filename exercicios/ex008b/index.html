Código-fonte / Pré-Formatado

O comando document.getElementById('teste') é escrever em linguagem javascript

Aqui temos um Parágrafo de código, sem o <pre>:
num = int(input("Digite um número: ")) if num %2 == 0: print("O número é par") else: print("O número é impar")

Já aqui, usando o <pre> (Pode variar caso a linha no VScode esteja perto ou longe do inicio): 

        
num = int(input("Digite um número: "))
    if num %2 == 0:
        print("O número é par")
    else:
        print("O número é impar")
